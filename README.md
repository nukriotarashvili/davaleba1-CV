# CV/Resume - Interactive Portfolio

მოდერნიზებული, ინტერაქტიული CV/Resume ვებ-გვერდი dark mode, რესპონსიული დიზაინით და ფოტოს ატვირთვის ფუნქციონალით.

ფუნქციები:

 - Dark/Light Mode - თემის გადართვა toggle ღილაკით
 - დაცულია რესპონსიულობა
 - პროფილის ფოტოს ატვირთვა და localStorage-ში შენახვა
 - პროგრეს ბარები ანიმაციით
 - ინტერაქტიული ელემენტები timeline-სა და წრეებზე. Hover ეფექტი
 - theme ინახება localStorage-ში


  ინსტალაცია

1. რეპოზიტორიის კლონირება:
```bash
git clone https://github.com/nukriotarashvili/davaleba1-CV.git
cd davaleba1-CV
```

2. ფაილების გახსნა:
  use "go live".


 ტექნოლოგიები

- HTML - სემანტიკური მარკაპი
- CSS - მოდერნული სტილები, CSS Variables, Flexbox, Grid
- JavaScript - DOM მანიპულაცია, LocalStorage API, FileReader API
